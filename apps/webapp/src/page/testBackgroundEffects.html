Background Effects V2 Browser Test

Note: This test requires camera access. When prompted, allow camera access to test the full functionality.

Query Parameters:
  • ?mode=blur - Effect mode: blur, virtual, or passthrough
  • ?debug=maskOverlay - Debug mode: off, maskOverlay, maskOnly, or edgeOnly
  • ?quality=A - Quality: auto, A, B, C, or D
  • ?blur=0.7 - Blur strength (0-1)
  • ?fps=30 - Target FPS
  • ?bg=gradient - Virtual background: gradient, grid, or none
Example: ?mode=blur&debug=maskOverlay&quality=A&blur=0.7&fps=30
